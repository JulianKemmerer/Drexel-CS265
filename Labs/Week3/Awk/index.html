    Some filters

    wc [-c] [-w] [-l] file
    word count. Counts the # of characters, words, or lines in file
    sort [-n] [-r] [-k start [, end]] file
    sorts input lexigraphically; -n numerically, -r reverse order, -k uses column(s) start..end
    grep [-E] regexp file(s)
    Searches for regular expressions (-E extended regular expressions) in files
    tr [-d] set1 [set2] file(s)
    translate - translates characters in set1 to corresponding character in set2, or -d deletes characters in set1
    awk program file(s)
    A full programming language, acts on each line of file(s) in order
